第三版

为了防止递归长时间占用线程，需要将整个渲染工作拆分成若干小的工作，每次浏览器空闲的时候都进行我们的工作。

React 解决这个问题的办法是创建了一种 fiber 数据结构，这种结构可以方便的链接父节点子节点兄弟节点，这样每个工作单元只处理个节点，然后下次在处理子节点， 如果没有子节点那么看看是否有兄弟节点，有的话就处理兄弟节点。
